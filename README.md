# Ultimate Todo App Pro

A **professional Todo App** with Edit, Login system, AI-powered suggestions, gamification, and streak tracking. Fully frontend, responsive, and portfolio-ready.

## Features

- User login & logout (tasks stored per username)
- Add/Edit/Delete tasks with title, description, due date, priority
- Subtasks support
- Search, filter, sort, clear completed
- Drag & drop for task ordering
- Dark/Light mode toggle
- Streak tracking with confetti reward
- AI-powered suggestions: top 3 tasks based on priority + due date
- Fully responsive

## Tech Stack

- HTML, CSS, JavaScript
- LocalStorage for user-specific data
- Frontend-only (ready for PWA upgrade)
